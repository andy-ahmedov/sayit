---
name: plan-pdf-tts-implementation
description: Use this skill when the task is to plan or restructure implementation work for this Russian PDF-to-audio repository before coding.
---

Read these files first:
- `README.md`
- `AGENTS.md`
- `docs/ARCHITECTURE.md`

Then produce a concrete plan with:
1. goal;
2. scope;
3. files to touch;
4. implementation order;
5. risks;
6. verification commands.

Do not start coding until the plan is coherent.
Keep the plan short and actionable.
